# Sleep vs Academic Performance — Data Analysis

## Overview 
Analysis of 10,000 high school students to investigate the relationship between sleep hours and academic performance. Found that previous academic scores showed correlation with performance (r = 0.92), while sleep hours showed almost no correlation (r = 0.048) and hours studied showed a moderate relationship (r = 0.37). Built this analysis using Python, Pandas, SQL/SQLite, and Tableau. 

## Tools Used
- Python & Pandas — data cleaning, EDA, analysis
- SQL / SQLite — database queries
- Tableau — interactive dashboard
- matplotlib & seaborn — visualizations

## Dataset
Kaggle — Student Performance (Multiple Linear Regression)
https://www.kaggle.com/datasets/nikhil7280/student-performance-multiple-linear-regression

## Key Findings
- Previous scores were by far the strongest predictor of performance (r = 0.92)
- Hours studied showed a moderate positive relationship (r = 0.37)
- Sleep hours showed almost no correlation with performance (r = 0.048)
- Sample question papers practiced had minimal impact (r = 0.043)
